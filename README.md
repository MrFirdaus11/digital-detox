# DigitalDetox — Hadir di Dunia Nyata

Aplikasi PWA timer fokus untuk membantu mengurangi screen time dan lebih hadir di dunia nyata.

## Fitur

- ⏱ Timer count-up & countdown (Pomodoro 15/25/45 mnt)
- 📊 Progress chart mingguan/bulanan (Chart.js)
- 🔥 Streak tracker & milestone badges
- 📝 Catatan sesi
- 🌙 Dark mode
- 🔊 Suara notifikasi
- 📲 PWA — installable, offline-ready
- 📥 Ekspor/Impor data JSON
- 📷 Ekspor grafik sebagai PNG

## Teknologi

- Vanilla JavaScript (modular pattern)
- CSS Custom Properties (design tokens)
- Chart.js
- Service Worker (PWA)
